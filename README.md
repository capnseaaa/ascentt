# Ascent

A soccer club-management game — pick a club, climb the pyramid, become champions.

## Run it locally

```
npm install
npm run dev
```

Then open the URL it prints (usually http://localhost:5173).

## Deploy for free (Vercel)

See the deployment steps your assistant gave you, or in short:

1. Push this folder to a new GitHub repo.
2. Go to vercel.com, sign in with GitHub, click "Add New Project", pick the repo.
3. Leave all settings on default (Vercel auto-detects Vite) and click Deploy.
4. You'll get a public `https://your-project.vercel.app` URL — share that with friends.

Progress saves in each player's own browser (localStorage), so everyone who visits the link plays their own separate career.
